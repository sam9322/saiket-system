## 🚀 Detailed Task Breakdown

### 📝 Task 1: Basic Blog Post Layout
- **Focus**: Pure Semantic HTML5.
- **Highlights**: Uses `<article>`, `<header>`, `<main>`, `<section>`, `<figure>`, `<figcaption>`, `<address>`, `<time>`, and `<footer>` tags to build a clean, accessible document structure.

### 💳 Task 2: Styling a Product Card
- **Focus**: HTML & CSS Styling.
- **Highlights**: Features a high-resolution studio headphone shot, rating stars, price discount badge, and an interactive "Add to Cart" button with smooth gradient glow animations and hover scale effects.

### 📐 Task 3: Responsive Webpage Layout
- **Focus**: CSS Grid Areas & Media Queries.
- **Highlights**: Implements a 4-section layout (Header, Sidebar, Main Content, Footer) using `grid-template-areas`. Automatically switches from a 2-column desktop grid to a single-column stacked layout on mobile screens (`< 768px`).

### ❓ Task 4: Interactive Quiz Application
- **Focus**: JavaScript DOM Manipulation & Validation.
- **Highlights**: Multi-choice quiz with real-time selection state, progress bar, live score counter, error shake feedback animation on skipped questions, and a final score summary screen.

### 📝 Task 5: Mini Project - TaskFlow To-Do List App
- **Focus**: Full CRUD JavaScript & LocalStorage Persistence.
- **Highlights**: Complete task manager allowing users to add, inline edit, delete, and toggle task completion. Includes active/completed category filters and automatic `localStorage` synchronization across reloads.

### 🌐 Task 6: Real-World Simulation - Landing Page Clone (AuraCloud AI)
- **Focus**: Advanced Responsive Design & JavaScript Interactivity.
- **Highlights**:
  - **Glassmorphism Header**: Mobile hamburger menu toggle drawer.
  - **Interactive Theme Customizer**: Real-time switching between *Midnight Indigo*, *Cyber Purple*, and *Electric Cyan* visual lighting themes.
  - **Dynamic Pricing Calculator**: Toggle between Monthly and Annual billing with 20% discount recalculations.
  - **Toast & Modal Systems**: Interactive form submission validation with toast popups and account activation modal dialogs.

## 🛠️ Technologies Used

- **HTML5**: Semantic elements, accessibility attributes, form validation.
- **CSS3**: Custom properties (variables), Flexbox, CSS Grid, Media Queries (`@media`), Keyframe Animations, Glassmorphism backdrop filters.
- **JavaScript (ES6+)**: Event listeners, DOM manipulation, state management, array functions, `localStorage` API, custom toast notifications.
